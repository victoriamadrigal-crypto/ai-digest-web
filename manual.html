Introducción

¿Qué es AI Masterclass?

AI Masterclass es una plataforma de aprendizaje diseñada para equipos mixtos (técnicos y no técnicos) que ya conocen las IAs pero quieren sacarles más partido. No es un curso teórico — todo está orientado a la práctica inmediata en el trabajo real.

🎓 11 lecciones base permanentes

Fundamentos y técnicas esenciales de Claude, Gemini y Kiro. Siempre disponibles, puedes volver a ellas cuando quieras.

⚡ 4 lecciones avanzadas nuevas cada semana

Generadas automáticamente cada lunes con técnicas avanzadas, casos de uso reales y ejemplos de prompts listos para copiar.

🎯 Reto semanal con ejercicios guiados

4 ejercicios prácticos cada semana con pasos detallados, prompts de ejemplo y criterios de éxito claros.

🧪 Sandbox para practicar en tiempo real

Prueba tus prompts directamente en la plataforma usando Groq (gratis) sin salir de la página.

Cómo empezar

Primeros pasos

Puedes empezar en 2 minutos sin configurar nada. Solo si quieres usar el Sandbox necesitarás una API key gratuita.

1

Abre el curso

Ve a la URL del curso. No necesitas cuenta ni registro — todo funciona directamente en el navegador.

2

Elige un módulo

Usa la barra de navegación superior para ir a Claude, Gemini o Kiro. Haz clic en cualquier lección para expandirla.

3

Lee y practica

Cada lección tiene teoría, ejemplos de prompts con botón "Copiar" y ejercicios. Responde los quizzes y completa los ejercicios de práctica.

4

Marca las lecciones completadas

Al terminar cada lección, haz clic en "Marcar como completada". Tu progreso se guarda automáticamente en el navegador.

5

Explora el Reto Semanal

Cada lunes aparecen 4 ejercicios nuevos en "🎯 Reto Semanal". Son ejercicios guiados paso a paso — perfectos para practicar en equipo.

Reto Semanal

Cómo funciona el Reto Semanal

Cada lunes a las 9h se generan automáticamente 4 ejercicios nuevos relacionados con un tema de IA. El objetivo es que el equipo los practique durante la semana y compare resultados.

🟢 Ejercicios Básicos (x2)

Para cualquier persona del equipo. Incluyen pasos muy detallados y un prompt completo listo para copiar. No requieren experiencia técnica.

🟡 Ejercicios Intermedios (x2)

Para quienes ya tienen algo de experiencia con las IAs. Requieren adaptar el prompt y tomar decisiones sobre cómo enfocar la tarea.

✅ Cómo sacarle partido en equipo

Dedica 15 minutos a la semana a hacer al menos un ejercicio. Luego comparte el resultado con un compañero y comparad los outputs — las diferencias de enfoque son muy reveladoras.

🗓️ Rotación de temas

El sistema rota entre 25 temas diferentes de forma automática — nunca se repite el mismo tema en menos de 6 meses. Los temas cubren desde prompting avanzado hasta detección de alucinaciones, pasando por casos de uso por rol.

Lecciones Extra

Lecciones avanzadas semanales

Cada semana se generan 4 lecciones extra de nivel avanzado sobre el tema de la semana. Cada lección incluye teoría, comparación de prompts (malo vs bueno) y un ejercicio de práctica guiado.

📖 Estructura de cada lección extra

Teoría — Explicación del concepto con analogía del mundo real y cuándo usarlo.

Ejemplo real — Comparativa de cómo NO y cómo SÍ hacer la tarea, con prompts completos.

Práctica — Ejercicio guiado con prompt listo para copiar y pasos detallados.

📚 Histórico completo

Todas las lecciones generadas se guardan en el histórico. Puedes acceder a cualquier semana anterior desde el botón "Ver histórico completo" en la sección de lecciones extra.

Sandbox

Cómo usar el Sandbox

El Sandbox te permite probar prompts en tiempo real usando Groq (gratuito). Necesitas configurar tu API key una sola vez.

1

Obtén tu API key de Groq (gratis)

Ve a console.groq.com → crea una cuenta → API Keys → Create API Key. La key empieza por gsk_

2

Configúrala en el Sandbox

En la sección Sandbox, pega tu key en el campo "Configura tu API key de Groq" y haz clic en Guardar. Solo lo harás una vez — se guarda en tu navegador.

3

Escribe o carga un prompt

Puedes escribir tu propio prompt o usar uno de los 6 prompts de inicio rápido para empezar. Opcionalmente añade un contexto del sistema.

4

Envía y analiza la respuesta

Haz clic en "Enviar prompt". Verás la respuesta junto con el modelo usado, los tokens consumidos y el tiempo de respuesta. Puedes guardar la respuesta como nota.

🎛️ Modelos disponibles

Llama 3.1 8B — Más rápido, ideal para pruebas rápidas · Llama 3.3 70B — Más potente, mejor calidad · Mixtral 8x7B — Mejor para textos largos y contexto amplio

Progreso

Mi progreso y exportación

La sección "Mi progreso" muestra cuántas lecciones has completado, el porcentaje por herramienta y las notas guardadas del Sandbox.

💾 Dónde se guarda tu progreso

Todo se guarda en el localStorage de tu navegador — no necesitas cuenta ni servidor. Esto significa que tu progreso está ligado al navegador y dispositivo que uses. Si cambias de navegador o limpias los datos del navegador, el progreso se reinicia.

📄 Exportar a PDF

Desde "Mi progreso" puedes exportar un PDF con tus lecciones completadas, el porcentaje por herramienta y las notas del Sandbox. Útil para revisiones de equipo o para compartir tu avance con tu manager.

👥 Para managers de equipo

Pide a cada miembro del equipo que exporte su PDF de progreso una vez al mes. Es una forma sencilla de ver quién está avanzando y en qué herramientas necesita más apoyo — sin dashboards ni sistemas complejos.

FAQ

Preguntas frecuentes

¿Tiene algún coste usar la plataforma? ⌄
La plataforma en sí es completamente gratuita. El Sandbox usa la API de Groq, que también tiene un tier gratuito más que suficiente para uso normal (1.500 peticiones/día). Solo pagarías si superases ese límite, lo cual es muy difícil con uso habitual.
¿Con qué frecuencia se actualiza el contenido? ⌄
Cada lunes a las 9h se generan automáticamente: el Tip de la semana, 4 lecciones avanzadas nuevas y 4 ejercicios del Reto Semanal. Las 11 lecciones base son permanentes y no cambian.
¿Puedo usar el curso en el móvil? ⌄
Sí, la plataforma es responsive y funciona en móvil. Para el Sandbox es más cómodo usar un ordenador ya que necesitas escribir prompts largos, pero las lecciones y retos se ven perfectamente en cualquier dispositivo.
¿Mi progreso se comparte con el resto del equipo? ⌄
No. El progreso se guarda solo en tu navegador y es completamente privado. Para compartirlo con tu equipo o manager, usa el botón "Exportar PDF" en la sección "Mi progreso".
¿Qué pasa si el contenido semanal no se carga? ⌄
El contenido semanal (Reto, Tip y Lecciones Extra) se genera cada lunes mediante un proceso automático en GitHub Actions. Si no se carga, puede ser que el proceso aún no haya corrido esa semana o que haya habido un error. Las 11 lecciones base siempre están disponibles independientemente de esto.
¿Puedo sugerir temas o mejoras? ⌄
La plataforma está alojada en GitHub — puedes abrir un issue en el repositorio con tu sugerencia. También puedes hablar con la persona que administra el repo en tu equipo para que añada o cambie contenido.
¿Necesito tener cuentas en Claude, Gemini y Kiro? ⌄
Para las lecciones teóricas no necesitas nada. Para practicar los ejercicios sí necesitarás acceso a las herramientas. Claude y Gemini tienen versiones gratuitas en claude.ai y gemini.google.com. Kiro es gratuito en su fase de preview en kiro.dev.
